## API Endpoints

All endpoints are live and working:

### GET `/health`
```bash
curl http://localhost:8001/health
```

Response:
```json
{
  "status": "ok",
  "chatbot": "Secutix Documentation Chatbot",
  "pages_indexed": 2487,
  "languages_supported": ["English", "French", "Spanish", "German"],
  "auth_method": "OAuth SubToken (sk-ant-oat-*)",
  "rag_model": "claude-sonnet-4-6",
  "embedding_model": "BAAI/bge-small-en-v1.5"
}
```

### GET `/search?q=<query>&lang=<lang>`
Semantic search without LLM (instant):
```bash
curl "http://localhost:8001/search?q=S-360%20features&lang=en"
```

### POST `/query`
Full RAG with answer generation:
```bash
curl -X POST http://localhost:8001/query \
  -H "Content-Type: application/json" \
  -d '{
    "question": "What is new in S-360 v4?",
    "language": "en"
  }'
```

Response includes:
- `answer`: Full LLM-generated answer (PM persona)
- `sources`: Array of 1-3 documents with links and relevance scores
- `detected_language`: Auto-detected language
- `num_sources`: Number of documents used

### POST `/summarize`
Structured formatted response:
```bash
curl -X POST http://localhost:8001/summarize \
  -H "Content-Type: application/json" \
  -d '{
    "question": "Release notes for latest version",
    "language": "en"
  }'
```

## RAG Pipeline

### Document Selection
- **Source**: Search top 10 results from Pinecone
- **Filter**: Keep only >50% relevance
- **Cap**: Maximum 3 documents
- **Goal**: Focus on highest-quality sources

### Answer Generation
- **Persona**: Senior Product Manager at Secutix
- **Model**: claude-sonnet-4-6
- **Context**: 3 documents + system prompt
- **Output**: Professional, attributed answer

### Source Attribution
Every response includes:
- **Title**: Exact page title
- **URL**: Direct Confluence link
- **Relevance**: Match percentage
- **Snippet**: First 200 chars preview

### Language Support
- **Input**: Auto-detect (English, French, Spanish, German)
- **Search**: In user's language (with fallback)
- **Output**: Answer in user's language
- **Zero translation overhead**: Native multilingual indexing

## Troubleshooting

### Backend won't start
```bash
# Check Python version
python3 --version  # Should be 3.8+

# Check dependencies
pip install -r requirements.txt

# Check environment
cat .env | grep ANTHROPIC_SUBTOKEN  # Should not be empty
```

### Pinecone connection fails
```bash
# Verify API key
curl -H "Api-Key: $PINECONE_API_KEY" \
  https://api.pinecone.io/indexes

# Check index exists
python3 -c "from pinecone import Pinecone; pc = Pinecone(api_key='...'); print(pc.Index('secutix-docs').describe_index_stats())"
```

### API calls return 401
```bash
# Token might have expired, regenerate:
claude setup-token

# Copy the new token to .env:
ANTHROPIC_SUBTOKEN=sk-ant-oat01-...

# Restart backend
python3 backend.py
```

### LLM answers are slow
- First request: ~2-3 seconds (model loading)
- Subsequent: ~1-2 seconds (cached)
- Search fallback: <100ms (instant)
