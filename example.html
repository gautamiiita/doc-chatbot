📚 Secutix Documentation Chatbot

AI-powered Q&A over your Release Notes. Multilingual support. Easy to embed.

⚡ Instant Answers

Ask questions about Secutix features and get instant answers powered by Claude AI and semantic search.

🌍 Multilingual

Supports English, French, German, Spanish, and more. Ask in any language, get answers in your language.

📖 Source Attribution

Every answer includes links to the relevant documentation pages with relevance scores.

📱 Responsive Design

Works on desktop, tablet, and mobile. Beautiful, modern chat interface that matches your brand.

🔒 Privacy First

Run on your own server. Control data. GDPR compliant. No tracking.

⚙️ Easy Integration

Copy 3 lines of code. Works with any website, framework, or CMS.

Quick Integration

1. Install React Component

import SecutixChatbot from './widget.jsx'; import './widget.css';

2. Add to Your App

<SecutixChatbot apiUrl="https://your-api.com" theme="light" />

3. That's it!

The chatbot will appear as a floating button in the bottom-right corner of your page.

API Endpoints

Query Documentation

POST /query { "question": "What's new in Zumstein V4?", "language": "en", "top_k": 5 }

Vector Search Only

GET /search?q=installation

List All Pages

GET /pages

Health Check

GET /health
